Contributions

These page contains links to programs and other contributions that do not belong into our CVS. Although applications found here are reported to work they are not supported by the Fresco developers. Please report any problem with anything you find here to the authors directly!

  • CD player and Mixer written by Phil and Paul. These were the first contributed applications. Don't they just look great;-)
  • French translations This site has the french translations (we try to keep those on this page in sync with those). There also is a simple client using a rudimentary QT-ish wrapper around the Fresco API.
  • Java clients Ben put his Java clients on the web (those from the screenshots page). Once they build in a free environment and some design issues are fixed these will become the basis for the Fresco Java client demos.
  • Fresco (Berlin) RPMs on russian site The russian ALT Linux Team has a RPM for their Mandrake-based Linux distribution. A RPMS is available too for those people who want to build their own RPMs. I had trouble downloading the files with Mozilla, but I was able to retrieve them with a normal (passive) FTP client.
