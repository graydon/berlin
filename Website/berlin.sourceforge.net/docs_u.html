Documentation for Users

Old Fresco Tutorial

The very first thing you should read is the Vintage Fresco Tutorial. The terminology has drifted a little meanwhile, but there is still no single piece of documentation we've written which describes the system better to a newcomer.

Fresco Wiki

We have put together a Wiki for discussing and posting questions about Fresco. Please feel free to modify it and extend it. It's a very easy way to get answers to questions, and build a knowledge base while we're at it.

In particular, you may want to check out the Fresco FAQ.
