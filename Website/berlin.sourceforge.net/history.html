Fresco's history

The history of this project is quite interesting, and may look different depending on whom you ask. So here is our view on it:

1988 : Publication:Applying Object-Oriented Design to Structured Graphics pdf
Implement a 2D structured graphics library presenting an OO model to the programmer. Structured graphics is useful for building applications that use a direct manipulation metaphor. Object-oriented languages offer inheritance, encapsulation, and runtime binding of operations to objects.
1989 : Publication:Unidraw: a Framework for Building Domain Specific Graphical Editors
1990 : Publication:Glyphs: Flyweight Objects for User Interfaces
1993 : Fresco: make the IV API CORBA aware, i.e. add distributed graphical embedding to the capabilities of IV.
1994 : Release:ivtools 0.3: a set of editors built with the Unidraw Framework
1995 :
Fresco proposed at OMG to RFP for Compound Document Facilities goal: combine Fresco structured graphics with OpenDoc embedding and document management
