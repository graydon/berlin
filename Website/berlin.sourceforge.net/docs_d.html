Documentation for Developers

Latest Fresco Tutorial

Documentation is slowly coming together for what code has been written so far. If you would like to contribute to the documentation you're welcome to do so -- it's available as DocBook (XML) in our CVS Repository, as well as in HTML form. A french translation of the HTML-version is available too. This translation must be reviewed: It is considered to be beta.

Coding Style Guide

We have come up with a Coding Style Guide to help us keep our code consistent across files. Please read it early and read it often and help to fix files not yet conforming to the standard specified in this document.
